# 🩺 Skin Disease Detection App (Android Frontend)

## 📱 Project Overview
This is a Kotlin/XML based Android frontend application designed for a Skin Disease Detection system. The app provides a complete user flow from onboarding to results and recommendations. It is currently the frontend stage, with AI/ML backend integration planned in future.

---

## 🧭 App Flow / Screens

The application follows a structured multi-screen workflow:

1. Splash Screen  
2. Disclaimer Screen  
3. Language Selection Screen  
4. Patient Information Screen  
5. Dashboard Screen  
6. Scan Screen (Image Upload)  
7. Symptoms Input Screen  
8. Analyzing Screen  
9. Results Screen  
10. Recommendations Screen  
   - Medical Recommendations  
   - Organic/Home Remedies  
11. View History Screen  
12. Delete History Functionality  

---

## ✨ Key Features
- Multi-step guided user flow
- Clean and user-friendly UI design
- Image scanning interface (frontend)
- Symptoms input system
- Result and recommendation display system
- History management (view & delete)
- Structured navigation between screens
